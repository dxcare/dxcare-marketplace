# 테마 레퍼런스

자동 생성 문서 — `scripts/theme-catalog.mjs --format=md` 출력.
단일 출처는 `_templates/theme/*.json` 16개 프리셋이다.

## 테마 선택 가이드

| 카테고리 | 프리셋 |
| --- | --- |
| 투자/IR (investor) | `apple-keynote`, `corporate-slate`, `keynote-dark`, `midnight-jewel` |
| 테크/개발 (tech) | `cyber-gradient`, `electric-ink`, `neon-terminal`, `stormy-night` |
| 교육/헬스케어 (education) | `paper-editorial`, `soft-botanical`, `warm`, `warm-earth` |
| 크리에이티브 (creative) | `bento-pastel`, `retro-wave` |
| 기본/범용 (basic) | `corporate`, `minimal` |
